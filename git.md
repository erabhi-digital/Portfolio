# Git Daily Commands

## Clone Repository

```bash
git clone https://github.com/user/repo.git
```

## Check Status

```bash
git status
```

## Add Changes

```bash
git add .
```

## Commit Changes

```bash
git commit -m "Add new feature"
```

## Push Changes

```bash
git push origin main
```

## Pull Latest Changes

```bash
git pull origin main
```

## Create New Branch

```bash
git checkout -b feature-login
```

## Switch Branch

```bash
git switch main
```

## Merge Branch

```bash
git merge feature-login
```

## View Commit History

```bash
git log --oneline
```
